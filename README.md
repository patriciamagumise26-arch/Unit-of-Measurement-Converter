# Unit-of-Measurement-Converter

This is a unit converter available in two versions:  a web application for instant browser based conversions and a Python command line tool for desktop use.

Web Version

Try it now:

Features
- Instant conversions in your browser
- Mobile-friendly responsive design
- No installation required
- Real-time conversion between inches, feet, and yards

How to Use (Web Version)
1. Visit the link above
2. Select your starting unit (inches, feet, or yards)
3. Enter the value you want to convert
4. Select the target unit
5. Click "Convert" to see the result instantly! 


Python Command Line Version

A command-line Unit Converter built using Python for desktop users who prefer terminal-based tools.


Technologies Used
- Python 
- While loops
- Conditional statements (`if / elif / else`)
- User input handling

How to Use (Python Version)

Option 1: Run the Python Script
1. Clone the repository:
  
   git clone https://github.com/patriciamagumise26-arch/Unit-Converter.git
   cd Unit-Converter
   

2. Ensure Python is installed:
   
   python --version
   
(If not installed, download from [python.org](https://www.python.org/downloads/))

3. Run the program:
  
   python Unit_of_measurement.py
   

Option 2: Use the Windows Executable
1. Download `Unit_of_measurement.exe` from the repository
2. Double-click the file to run
3. No Python installation required!  (Windows only)
